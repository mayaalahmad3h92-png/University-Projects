# 📚 University Projects System

## Rules
- Folder: `projects/semester/` or `projects/graduation-1/` or `projects/graduation-2/`
- Filename: `XXX-project-name.pdf` (3 digits + name)
- Example: `projects/graduation-1/042-ai-chatbot.pdf`

## How to Submit
1. Fork this repo
2. Upload your file to correct folder
3. Create Pull Request
